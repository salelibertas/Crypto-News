TSF plans HARD FORK for 1.12

Written by João José. November 28, 2021
tranding

The hard fork that we are going to perform on the TSF network will not only increase the block gas limit & block size but also enhance the transaction processing speed.

Performing hard fork offers several benefits to end-users. We are going to apply the same at TSF Node to increase:
• The block gas limit & block size from 4.7 to 12.5 million.
• Up to 600 transactions per block.
